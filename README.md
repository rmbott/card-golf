# card-golf
A card game (lowest score wins, hence the name "golf").

## Initial Setup
Install poetry.
In project folder
- run `poetry install` to install dependencies.
- run `poetry shell`
- run `docker compose up --build`
Finally, point your browser to `http://localhost:8000`.

## Usage
In project folder
- run `poetry shell`
- run `docker compose up`
Finally, point your browser to `http://localhost:8000`.
